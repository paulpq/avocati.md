BAA Ariadna Dodi & Gheorghe Iudin — Website
============================================

FOLDER STRUCTURE
----------------
project/
├── pages/        HTML pages (index, despre, servicii, onorarii, avocati, contact)
├── css/          styles.css
├── js/           main.js, i18n.js
├── assets/       logo.png
└── README.txt

The home page is pages/index.html.

TO PREVIEW / EDIT
-----------------
Open pages/index.html in a browser. All pages reference the CSS, JS and
logo using relative paths (../css/, ../js/, ../assets/), so keep the
folder structure intact.

TO GO LIVE
----------
Upload the entire project/ folder to your web host and point the domain
at pages/index.html (or set pages/ as the web root, depending on your
host's configuration).

LANGUAGES
---------
Switch between RO / RU / EN / IT using the buttons in the top-right
corner of any page.
